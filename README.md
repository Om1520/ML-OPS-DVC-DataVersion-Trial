**# ML-OPS-DVC-DataVersion-Trial**
This project decouples source code from data storage. This project demonstrates how to decouple source code from data storage. While Git tracks the code and .dvc metadata files, DVC manages the actual data transitions, enabling:  
**Reproducibility:** Switching between different versions of a dataset as easily as switching Git branches. **Storage Efficiency:** Tracking changes in data without bloating the Git repository.                       
**Data Provenance:** Maintaining a clear audit trail of which dataset version was used to train a specific model training run.
